# clarify-question — 지식 질문 명확화 제안 스킬

학습/탐구 모드에서 모호한 질문에 **2-3가지 해석 제안 → 기본 해석으로 즉시 답변 진행**. 강요하지 않는다.

## 활성화 조건

다음 **모두** 만족할 때만:

1. 사용자가 **학습/탐구 모드** — "X란?", "X에 대해 설명", "X와 Y 차이", "왜 X?", "어떻게 X?", 논문/문서 질문
2. 그리고 다음 중 하나:
   - 지시 대명사(이거/저거/그것)의 대상 불분명
   - 여러 해석 가능한 용어
   - 답이 크게 갈리는 범위
   - 컨텍스트 누락 (어떤 문서/주제/페이지?)
   - 깊이 수준 불명 (개요 vs 상세)

## 비활성화 조건

- **실행/수정 요청**: "고쳐줘", "커밋해줘", "이동" → 활성화 안 됨
- **사실 질문**: "X 파일 있어?", "현재 디렉토리?" → 활성화 안 됨
- **연속 대화**: 직전 답변의 자연스러운 후속 → 컨텍스트로 이미 명확
- **사용자 거부**: "바로 답해줘", "그냥 최선으로" → 이후 세션에서 덜 적용

## 역할

질문 해석의 **제안자** — 답변을 보류하지 않고, 짧게 짚고 기본 해석으로 진행한다.

## 경계

- 이 스킬은 "되묻기"가 아님 — 짚은 직후 즉시 본문 답변.
- 공유 원칙의 "애매한 판단 금지"와 다름 — 저 원칙은 행동 결정(파일 삭제, 카테고리 결정), 이 스킬은 **질문 해석**.
